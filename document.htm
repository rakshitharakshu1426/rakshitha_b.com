Power BI tools and deployment options

Tools and components for authoring, deploying, and extending Power BI across cloud and on-premises environments.

Power BI Desktop

Create interactive reports and semantic models with this free Windows application.

Power BI Report Builder

Author paginated reports with tables, charts, and parameters using this free Windows application.

Power BI Report Server

Deploy reports on-premises with a web portal for interactive and paginated reports.

Power BI Mobile

View and interact with reports on iOS, Android, and Windows devices.

On-premises data gateway

Keep data fresh by connecting the Power BI service to your on-premises data sources.

Power BI Embedded

Embed interactive reports in your applications using REST APIs and client SDKs.

Azure Analysis Services

Enterprise-grade analytical data models in the cloud, compatible with Power BI semantic models.

SQL Server Reporting Services

On-premises reporting platform for paginated reports and mobile reports.

DAX reference

Data Analysis Expressions (DAX) formula language for calculations in Power BI, Analysis Services, and Power Pivot.
